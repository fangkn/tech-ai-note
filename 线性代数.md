# 学习大纲

1、 向量的定义与运算

 - 向量的几何意义（方向 + 大小）
 - 向量加法与数乘（图形演示）
 - 向量的点积（内积）与夹角关系
 - 向量的长度、正交、单位向量
2、 线性组合与线性相关

 - 向量的线性组合是什么
 - 向量是否能互相“生成”空间
 - 线性无关 / 相关的意义

3、 空间与基（Basis）

 - 什么是“基”与“坐标系”
 - 不同基下的向量表示（坐标变换直觉）

4、矩阵的定义

 - 矩阵作为“数组”和“变换”的两种理解
 - 行、列、形状（shape）

5、矩阵的基本运算

 - 加法、数乘
 - 矩阵乘法的几何意义（旋转、投影、缩放）
 - 单位矩阵与零矩阵
 - 转置矩阵

6、线性方程组与矩阵

 - 矩阵形式：Ax = b
 - 线性方程的几何解释
 - 高斯消元法

7、矩阵的秩（Rank）

- 行空间、列空间
- 秩的意义：信息量 / 独立维度
- 秩与线性方程解的关系

8、可逆矩阵与行列式

 - 可逆的直觉：是否能“反变换”
 - 行列式的几何意义（面积/体积）

9、特征值与特征向量

 - 什么是特征值、特征向量
 - 线性变换中的“稳定方向”
 - 对角化的意义

10、对称矩阵与正交矩阵

 - 对称矩阵的性质（在机器学习中很常见）
 - 正交矩阵 = “纯旋转”
 - 正交投影

11、PCA（主成分分析）与降维
12、 神经网络中的矩阵乘法
13、线性回归的矩阵形式
14、奇异值分解（SVD）与数据压缩
15、向量空间模型（文本表示 / embedding）
